# QQBottom-master
低仿QQ底部按钮动画

最近看了QQ新出的版本，底部按钮的动画很有意思，所以自己就写了一个，不过是低配版，有很多需求优化的地方，
比如：QQ按钮在获取到焦点的时候，按钮会有一个移动的特效，这些就留给同学们自己去写了，哈哈。当然我也会自己慢慢的优化。
